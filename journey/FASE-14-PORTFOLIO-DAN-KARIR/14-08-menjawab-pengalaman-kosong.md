# 14-08: Menjawab Pengalaman Kosong

> **Fase**: 14 — Portfolio & Karir  
> **Prasyarat**: 14-07-linkedin-personal-branding  
> **Waktu baca**: 30-40 menit  
> **Kata kunci**: pengalaman kosong, gap, fresh graduate, career change, honesty, framing

---

## 📋 Ringkasan

Banyak junior khawatir soal "pengalaman kosong" di CV. Sebenarnya, yang dilihat adalah **skill dan attitude**, bukan tahun pengalaman.

**Target pemahaman:**
- Kamu bisa jawab soal gap pengalaman
- Kamu bisa frame pengalaman non-kerja
- Kamu bisa ubah kekurangan jadi kelebihan

---

## 1. Pertanyaan Interview

```
Q: "Kamu belum punya pengalaman kerja?"
```

### 1.1 Jawaban: Fresh Graduate

> "Saya fresh graduate, tapi selama kuliah saya fokus belajar
> Laravel dan mengerjakan project e-commerce. Project ini
> mencakup MVC, Service Layer, integrasi payment Midtrans,
> dan feature testing. Saya yakin skill teknis ini relevan."

### 1.2 Jawaban: Career Switch

> "Saya pindah dari [bidang lama]. Selama transisi, saya
> belajar intensif Laravel selama 6 bulan dengan project
> e-commerce. Saya terbiasa belajar cepat — terbukti dari
> [prestasi di bidang lama]."

### 1.3 Jawaban: Gap (1-2 tahun tidak kerja)

> "Saya mengambil waktu untuk fokus belajar pemrograman
> secara intensif. Hasilnya: saya menguasai Laravel + PHP
> dengan project portfolio yang bisa dilihat di GitHub saya."

---

## 2. Framing

| Kekhawatiran | Frame Positif |
|-------------|---------------|
| Tidak ada pengalaman kerja | "Project portfolio menunjukkan kemampuan teknis saya" |
| Belum pernah kerja tim | "Saya terbiasa pakai Git, paham code review" |
| Belum pernah production | "Saya paham deployment workflow, env config" |
| Belum pernah handle error | "Saya punya debugging strategy: log, Telescope, test" |

---

## 3. Apa yang Bisa Ditunjukkan

Meski tanpa pengalaman kerja formal:

```text
✅ GitHub portfolio yang rapi
✅ Codebase yang terstruktur (MVC, Service Layer)
✅ Testing (feature test)
✅ Git commit yang rapi
✅ Kemampuan belajar mandiri
✅ Problem-solving skill
```

---

## 🧪 Latihan

1. **Tulis jawaban.** Tulis jawaban untuk pertanyaan: "Kenapa kami harus hire kamu?"

2. **Frame ulang.** Ambil 3 kekurangan dalam CV-mu. Tulis frame positif untuk masing-masing.

3. **Praktik.** Minta teman interview mock. Jawab pertanyaan soal pengalaman kosong.
