# 14-09: Roadmap Menjadi Junior

> **Fase**: 14 — Portfolio & Karir  
> **Prasyarat**: 14-08-menjawab-pengalaman-kosong  
> **Waktu baca**: 35-45 menit  
> **Kata kunci**: roadmap, junior developer, career path, next steps, continuous learning

---

## 📋 Ringkasan

Setelah menguasai journey ini, kamu sudah setara junior dengan 6-12 bulan pengalaman. Tapi perjalanan tidak berhenti di sini — ini roadmap untuk terus berkembang.

**Target pemahaman:**
- Kamu paham posisi saat ini
- Kamu tahu langkah selanjutnya
- Kamu bisa prioritaskan belajar

---

## 1. Posisi Saat Ini

```
Junior Laravel Developer
├── Bisa baca dan pahami codebase existing
├── Bisa buat fitur CRUD sederhana
├── Paham MVC, Service Layer, Policy
├── Bisa nulis feature test
├── Paham Git, Composer, Artisan
└── Bisa deploy ke server

Target role:
- Junior Web Developer
- Junior Backend Developer
- Junior Laravel Developer
- Fresh Graduate PHP Developer
```

---

## 2. Technical Roadmap

### 2.1 Jangka Pendek (1-3 bulan pertama kerja)

```text
✅ Kuasai codebase perusahaan
✅ Paham workflow tim (Git flow, code review, deploy)
✅ Bisa handle bug fixing
✅ Baca dokumentasi Laravel lebih dalam
✅ Belajar queue (job, listener)
```

### 2.2 Jangka Menengah (3-12 bulan)

```text
📚 API Development (Laravel Sanctum, API resource)
📚 Livewire (frontend dinamis tanpa JS framework)
📚 Advanced Eloquent (subquery, union, raw)
📚 Testing (TDD: test dulu baru kode)
📚 Docker untuk development
```

### 2.3 Jangka Panjang (1-2 tahun)

```text
🎯 Vue.js / React (frontend framework)
🎯 CI/CD (GitHub Actions, Forge)
🎯 Cloud (AWS, DigitalOcean)
🎯 Performance optimization (caching, indexing)
🎯 Architecture (DDD, CQRS, Event Sourcing)
```

---

## 3. Non-Technical Skills

```text
🗣️ Komunikasi — jelaskan teknis ke non-teknis
🤝 Teamwork — code review, pair programming
⏰ Time management — estimasi deadline
📝 Dokumentasi — README, wiki, technical spec
🔍 Problem solving — debugging, root cause analysis
```

---

## 4. Estimasi Gaji (Indonesia, 2026)

| Posisi | Junior | Mid | Senior |
|--------|--------|-----|--------|
| **Laravel Developer** | 5-10 jt | 10-20 jt | 20-35 jt |
| **Fullstack Developer** | 5-12 jt | 12-25 jt | 25-40 jt |
| **Backend Developer** | 6-12 jt | 12-22 jt | 22-38 jt |

---

## 5. Target Mingguan

```text
Setelah dapat kerja, target mingguan:
- 1 PR merged (minimal)
- Baca 1 artikel teknis per hari
- 1 jam belajar hal baru (diluar kerja)
- Update catatan / journal
```

---

## 🧪 Latihan

1. **Buat roadmap pribadi.** Tulis rencana belajar 6 bulan ke depan. Topik apa saja?

2. **Cari role model.** Cari 3 developer senior di LinkedIn. Pelajari career path mereka.

3. **Set goals.** Tulis 3 goals karir: 1 bulan, 6 bulan, 1 tahun.

4. **Evaluasi.** Dari seluruh FASE 1-14, topik mana yang masih lemah? Prioritaskan belajar.
