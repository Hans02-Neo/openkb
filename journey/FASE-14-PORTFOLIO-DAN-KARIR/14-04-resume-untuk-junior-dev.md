# 14-04: Resume untuk Junior Dev

> **Fase**: 14 — Portfolio & Karir  
> **Prasyarat**: 14-03-readme-yang-menarik  
> **Waktu baca**: 35-45 menit  
> **Kata kunci**: resume, CV, curriculum vitae, junior, job application, ATS

---

## 📋 Ringkasan

Resume adalah **dokumen penjualan** — bukan riwayat hidup lengkap. Fokus pada skill yang relevan dengan posisi yang dilamar.

**Target pemahaman:**
- Kamu paham struktur resume junior
- Kamu bisa menulis bullet point yang efektif
- Kamu paham ATS (Applicant Tracking System)
- Kamu bisa buat resume dari codebase ini

---

## 1. Struktur Resume Junior

```text
[Nama]
[Email] [Phone] [LinkedIn] [GitHub]

RINGKASAN
2-3 kalimat: siapa, stack, goals.

SKILL TEKNIS
Bahasa: PHP, JavaScript, HTML, CSS
Framework: Laravel, Tailwind, Alpine.js
Tools: Git, Composer, NPM, Vite
Database: MySQL, SQLite

PENGALAMAN PROYEK
Nama Project | Tech Stack | 2026
- Fitur yang dibuat (gunakan action verbs)
- Tantangan dan solusi
- Link GitHub / demo

PENDIDIKAN
Nama Sekolah / Universitas | Tahun

SERTIFIKASI (opsional)
- Laravel Certification
- ...

BAHASA
- Indonesia (native)
- Inggris (passive / active)
```

---

## 2. Action Verbs

Gunakan **action verbs** untuk bullet point:

```text
✅ "Membangun fitur checkout dengan integrasi Midtrans"
✅ "Mendesain arsitektur MVC + Service Layer"
✅ "Menulis feature test untuk flow end-to-end"
✅ "Mengoptimasi query dengan eager loading (N+1 fix)"
✅ "Mengintegrasikan API RajaOngkir untuk shipping cost"

❌ "Bertanggung jawab atas..."
❌ "Membantu dalam..."
❌ "Mempelajari..."
```

---

## 3. Contoh Bullet Point dari Codebase

```text
Project: Olshop Koneksi (Laravel E-commerce)
├── Membangun arsitektur MVC dengan Service Layer
├── Mengintegrasikan Midtrans Snap untuk payment
├── Menulis feature test untuk checkout flow (15 test)
├── Mengimplementasikan cart session-based
├── Mendesain sistem authorization dengan Policy
└── Mengelola gambar produk dengan Spatie Media Library
```

---

## 4. ATS-Friendly

ATS (Applicant Tracking System) adalah software yang membaca resume secara otomatis.

```text
✅ Gunakan format PDF (bukan JPG/PNG)
✅ Gunakan font standar (Arial, Calibri)
✅ Gunakan section header standar (Experience, Education, Skills)
✅ Cantumkan tech stack secara eksplisit (Laravel, PHP, MySQL)
❌ Jangan pakai tabel, kolom, atau grafik
❌ Jangan pakai singkatan yang tidak umum
❌ Jangan pakai template complex
```

---

## 5. Tips Resume

```
✅ 1 halaman (max 2 untuk pengalaman banyak)
✅ PDF (pastikan bisa di-copy textnya)
✅ File name: Nama_Lengkap_Resume.pdf
✅ Cantumkan link GitHub + LinkedIn
✅ Proofread! (minta orang lain cek)
```

---

## 🧪 Latihan

1. **Buat draft resume.** Tulis resume 1 halaman berdasarkan codebase ini.

2. **Cek ATS.** Upload resume ke [jobscan.co](https://www.jobscan.co/) (free trial) untuk cek ATS score.

3. **Review.** Minta teman atau mentor cek resume. Feedback apa yang didapat?

4. **Bandingkan.** Cari contoh resume junior Laravel developer online. Bandingkan dengan punyamu.
