# 14-07: LinkedIn Personal Branding

> **Fase**: 14 — Portfolio & Karir  
> **Prasyarat**: 14-06-pertanyaan-interview-laravel  
> **Waktu baca**: 35-45 menit  
> **Kata kunci**: LinkedIn, personal branding, networking, profile, job search

---

## 📋 Ringkasan

LinkedIn adalah platform profesional untuk **personal branding** dan **job hunting**. Profile yang baik membantu recruiter menemukanmu.

**Target pemahaman:**
- Kamu bisa optimasi LinkedIn profile
- Kamu paham cara networking
- Kamu bisa apply job via LinkedIn

---

## 1. Profile Checklist

```text
□ Foto profesional (bukan selfie/liburan)
□ Headline deskriptif (bukan hanya "Mahasiswa")
□ About section dengan cerita
□ Experience dengan bullet point
□ Skills & endorsements
□ Sertifikasi (kalau ada)
□ Custom URL (linkedin.com/in/namamu)
□ 500+ koneksi (target)
```

### 1.1 Headline

```text
❌ "Mahasiswa Teknik Informatika"
✅ "Junior Laravel Developer | PHP | MySQL | E-Commerce"
```

### 1.2 About Section

```text
Saya adalah fresh graduate yang fokus di Laravel development.
Project terakhir: e-commerce app dengan Midtrans payment,
RajaOngkir API, dan feature testing.

Tech stack: PHP 8.3, Laravel 13, Tailwind, Alpine.js, MySQL.

Saya mencari opportunity sebagai Junior Laravel Developer.
Open to: Full-time, Remote/Hybrid.
```

### 1.3 Experience

```text
Project: Olshop Koneksi (Self-paced | Jan 2026 - Jun 2026)
├── Built e-commerce app with Laravel 13, PHP 8.3
├── Integrated Midtrans Snap payment gateway
├── Implemented MVC + Service Layer architecture
├── Wrote feature tests for checkout flow
└── Managed assets with Spatie Media Library
```

---

## 2. Networking

```text
✅ Connect dengan recruiter IT
✅ Follow perusahaan target
✅ Ikut grup Laravel / PHP Indonesia
✅ Comment di postingan tech
✅ Post tentang progress belajar

Jangan:
❌ Spam "Saya cari kerja" di postingan orang
❌ Kirim connection request tanpa note
```

---

## 3. Job Search

```text
Search keywords di LinkedIn:
- "Junior Laravel Developer"
- "Junior Backend Developer"
- "Junior Web Developer"
- "PHP Developer"

Filter: Entry Level, Remote/Hybrid, < 1 year experience.

Simpan search alert → email notification setiap ada job baru.
```

---

## 🧪 Latihan

1. **Cek profile.** Buka linkedin.com. Evaluasi profile kamu dari 1-10.

2. **Update headline.** Ubah headline sesuai format di atas.

3. **Tulis about section.** Tulis 3 paragraf tentang dirimu.

4. **Connect.** Cari 5 recruiter IT di LinkedIn. Kirim connection request dengan note personal.
